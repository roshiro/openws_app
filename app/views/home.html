Steps to get started

Saving a JSON object

        
  // JSON to be saved
  var product = {
    name: "Laptop",
    price: 2000
  };

  // Saving in a collection called "my_products"
  $.post("https://openws.herokuapp.com/my_products?apiKey=YOUR_API_KEY", product)
    .done(function(data) {
      console.log("Product saved successfully");
      ...
    });

Retrieving an object

        
  $.get("https://openws.herokuapp.com/my_products?apiKey=YOUR_API_KEY")
    .done(function(data) {
      console.log("Products retrieved.");
      ...
    });

Updating an object

Every object you save, receives an ID. This ID is saved in an attribute called "_id".

To update an object you need to pass in its ID in the URL, as shown below:

        
  var product = {
    name: "Apple Laptop",
    value: 2000
  };

  // In this example, the ID is 547ba6d00b07515f0d4b6c62
  $.put("https://openws.herokuapp.com/my_products/547ba6d00b07515f0d4b6c62?apiKey=YOUR_API_KEY", product)
    .done(function(data) {
      console.log("Product updated");
      ...
    });
