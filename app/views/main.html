How to start in 6 minutes

Follow the steps to use the API with AngularJS.

#1. Saving a JSON object

      ...
      var = taskJSON = {
        name: "My first task",
        completed: false
      };

      $http.post("https://openws.herokuapp.com/my_tasks", taskJSON)
        .success(function() {
          $log.debug("Task saved successfuly");
        });
      ...

#2. Retrieving all JSON objects

      ...
      $http.get("https://openws.herokuapp.com/my_tasks")
        .success(function() {
          $log.debug("Task saved successfuly");
        });
      ...
OpenWS API supports POST, PUT, GET and DELETE http methods. Check examples on how to use them here.
