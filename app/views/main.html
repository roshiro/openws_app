What's OpenWS

OpenWS makes web development faster by allowing webdevs to store their JSON objects into our platform. It means you don't need to worry about creating the server-side mechanism to make the hard work of saving, retrieving, searching, updating and deleting your data, we've done the hard work for you.

Our API supports POST, PUT, GET and DELETE. It means you will be allowed to save your data, update it, retrieve it and delete it.

Is OpenWS for me?

OpenWS is a great fit for your project if:

  • You're a web developer
  • Your project needs to store data
  • You have no knowledge of server-side development (if you do, that's fine)
  • You want to prototype and develop your front-end quickly

How to start in 6 minutes

Follow the steps to use the API with AngularJS.

#1. Create an account for free

You will receive an API KEY which will be used in the next step.

#2. Saving a JSON object

      
  ...
  var = taskJSON = {
    name: "My first task",
    completed: false
  };

  $http.post("https://openws.herokuapp.com/my_tasks?apiKey=YOUR_API_KEY", taskJSON)
    .success(function() {
      $log.debug("Task saved successfuly");
    });
  ...

#3. Retrieving all JSON objects

      
  ...
  $http.get("https://openws.herokuapp.com/my_tasks?apiKey=YOUR_API_KEY")
    .success(function() {
      $log.debug("Task saved successfuly");
    });
  ...
OpenWS API supports POST, PUT, GET and DELETE http methods. Check examples on how to use them here.
